# ts-networks Documentation

ts-networks is a system for building and running **propagator networks** — directed graphs where data flows through typed computation steps. It is designed to orchestrate agentic workflows: structured outputs from AI agents are routed, classified, and transformed through declarative pipelines that are easy to inspect, modify, and extend.

Programs are written in a small DSL, intended to be generated by language models and read by humans.

---

## Quick start

**1. Write a program** in a `.tsn` file. A minimal program defines a record, a function, and a network:

```
defrecord Point
  x: Number?;
  y: Number?;
end

defn distance
  signature: from [Point?(a), Point?(b)] to Number?;
  expression
    let dx = a.x - b.x;
    let dy = a.y - b.y;
    (dx * dx) + (dy * dy);
end

defnetwork distancePipeline
  signature: from [a, b] to result;
  propagate distance from [a, b] to result;
end
```

**2. Check it parses and type-checks:**

```bash
npx tsx scripts/check.ts my-program.tsn
npx tsx scripts/typecheck.ts my-program.tsn
```

**3. Run a network:**

```bash
npx tsx scripts/run.ts my-program.tsn distancePipeline 'a={x:0,y:0}' 'b={x:3,y:4}'
# result = 25
```

**4. Generate JSON Schemas for all records:**

```bash
npx tsx scripts/compile-schemas.ts my-program.tsn
```

**5. Inspect the AST:**

```bash
npx tsx scripts/parse.ts my-program.tsn | jq '.networks[0]'
```
