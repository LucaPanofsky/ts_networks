05

Legend

What each column means.

score
Hotspot rank 0–100 — where to look first. A weighted blend of churn, under-testedness, and blast-radius (fan-in), with size as a minor amplifier. Each input is normalized across the modules, so the score is relative to this codebase, not absolute.
src LOC
Lines of source in the module's .ts files (newline count, à la wc -l). Generated files (parser.js, *.d.ts) are excluded.
test LOC
Lines in the mirrored tests/ files for the module (plus any explicitly-attached cross-cutting test files).
test:src
test LOC ÷ src LOC — a rough test-investment proxy. Low can mean under-tested; high doesn't guarantee good tests (use cov for that).
cov
Statement coverage from jest, statement-weighted across the module's files (not an average of per-file percentages). ≥90% / ≥70% / <70%; — = not measured (run npm run analyze).
fanIn
How many other modules import this one — its blast-radius. High fanIn = a change here ripples widely.
fanOut
How many other modules this one imports — its outward coupling.
instab
Instability = fanOut ÷ (fanIn + fanOut), 0–1 (Martin's metric). 0 = stable (only depended-on); 1 = unstable (only depends on others). A stable module is costly to change; an unstable one is freer to.
commits
Number of git commits in history that touched any file in the module.
churn
Total lines added + deleted across all those commits — how much the module has moved over time.
risk
Count of weak-typing / unfinished markers: as any, : any, @ts-ignore, @ts-expect-error, TODO, FIXME, XXX.
kind
core algebra + central wiring · runtime DSL frontend, engine, sandbox, operations · tooling entrypoints/adapters.
⚠
Algebra surface (merge / I / naryUnpacking and friends) — given-and-correct; change only with explicit permission. Ranked but flagged, never a refactor target.

Codebase maintenance report — module = a first-level src/ subdirectory (loose root files are one-file modules). Generated by npm run analyze (repo_workspace/analysis/report.ts). ⚠ = algebra surface, off-limits without explicit permission.
